# A2.3.3 — Executive Decision Integration

Creates one authoritative decision state so executive reports cannot issue
contradictory instructions.

## Governed inputs

- Position and portfolio risk
- Investment Decision Support
- Portfolio Performance Positions
- Cost-Basis Coverage
- Portfolio Return Attribution

## Governed outputs

- Executive Decision State A233
- Report Action Cards A233
- Report Conflicts A233
- Report Data Readiness A233
- Report Integration Validation A233

## Core controls

- Critical portfolio risk blocks material accumulation.
- Stale prices block executable deployment recommendations.
- Current holdings are separated from external opportunities.
- Cost-basis and return-attribution coverage are explicit.
- Action cards include execution status, trigger, invalidation,
  confidence trend and risk impact.
- Executive CIO Report consumes the governed decision state.
