# A2.3.1.1 — Cost Basis Coverage and Attribution Eligibility

## Problem addressed

Missing cost basis was previously interpreted as zero, which caused the full
market value of affected positions to appear as unrealized profit.

## Policy

A position is attribution-eligible only when:

- Market Value > 0
- Cost Basis > 0

Otherwise:

- Gain/Loss is blank
- Return % is blank
- Contribution % is blank
- Eligibility Reason = `EXCLUDED — MISSING COST BASIS`

Portfolio-wide return is suppressed when cost-basis coverage is below 80%.

## Outputs

- Position Attribution Coverage A2311
- Account Attribution Coverage A2311
- Sector Attribution Coverage A2311
- Attribution Coverage Summary A2311
- Attribution Coverage Validation A2311
