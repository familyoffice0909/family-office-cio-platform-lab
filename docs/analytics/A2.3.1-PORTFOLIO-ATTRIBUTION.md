# A2.3.1 — Portfolio Attribution Engine

## Purpose

Adds governed position, account and sector attribution using the A2.3.0
architecture services.

## Source

`Portfolio Performance Positions`, produced by
`foRunPortfolioPerformance()`.

## Outputs

- Position Attribution A231
- Account Attribution A231
- Sector Attribution A231
- Attribution Executive Summary A231
- Attribution Validation A231

## Attribution basis

This release uses current unrealized gain/loss and cost basis. Time-series
performance attribution is reserved for a later wave.
