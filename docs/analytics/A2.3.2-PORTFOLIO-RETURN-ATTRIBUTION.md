# A2.3.2 — Portfolio Return Attribution

Adds period-over-period price-return attribution using consecutive portfolio snapshots.

A position is eligible only when it exists in both snapshots, quantity is unchanged, and beginning/ending effective prices are positive. Quantity changes are excluded because transaction cash-flow data is not yet available.

The first execution creates a baseline. The second and later executions create:

- Performance Snapshot A232
- Position Return Attribution A232
- Account Return Attribution A232
- Sector Return Attribution A232
- Return Attribution Summary A232
- Return Attribution Validation A232

Release target: v1.3.0.
